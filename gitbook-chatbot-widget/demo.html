🤖 GitBook Chatbot Widget

A lightweight, embeddable AI assistant for your documentation

👉 Look at the bottom-right corner! Click the chat button to try the widget.

Features

🎨 Customizable

Match your brand colors and style

⚡ Fast Search

Vector-powered semantic search

📱 Responsive

Works on all devices

🔌 Easy Integration

Just copy & paste

Quick Start

Add this code to your GitBook custom scripts section:

<script src="config.js"></script> <script src="chatbot-widget.js"></script> <link rel="stylesheet" href="chatbot-widget.css">

Try It Out

Click the chat button in the bottom-right corner and try asking:

  • "How do I configure authentication?"
  • "Tell me about fleet management"
  • "What is yard management?"
  • "How do I track vehicles?"

Configuration

Customize the widget by editing config.js:

const CHATBOT_CONFIG = { apiEndpoint: 'https://your-api.com/v1/search', primaryColor: '#0066cc', position: 'bottom-right', title: 'Documentation Assistant', maxResults: 5, autoOpen: false };

API Status

Backend: http://localhost:8001

Make sure your FastAPI server is running on port 8001 with CORS enabled.
